
** Setup:

#+begin_src sh
sudo mv /etc/nixos/hardware-configuration.nix{,.bak}
sudo mv /etc/nixos/configuration.nix{,.bak}

sudo ln -s $(readlink -f ./machines/$(hostname)/configuration.nix) /etc/nixos/configuration.nix
#+end_src
